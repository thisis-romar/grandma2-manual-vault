# Delete MA Network Switch from the network configuration

Use these steps to delete the MA Network Switch from the network configuration.

1.  Open the MA Network Configuration.  
    \- Press Setup, and tap under **Network** MA Network Configuration.  
    The MA Network Configuration window opens.
2.  Tap at Network Switch.
    
    ![](../../img/menu_ma-network-configuration_network-switch_v3_2.png)
    
    _MA Network Configuration window_
    
3.  Tap Delete.

The MA Network Switch is deleted from the network configuration.
