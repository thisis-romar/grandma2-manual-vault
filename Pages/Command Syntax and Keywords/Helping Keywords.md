# Helping Keywords

Helping keywords are used to create a relation between functions and objects. 

**Important:**  
Helping keywords are the prepositions and conjunctions of the console, so to speak. 

### Example:

**Helping keyword At:**

\[Channel\]> Copy Cue 3 **At** Cue 5

**Helping keyword Thru:**

 \[Channel\]> Delete Effect 4 **Thru** 6
