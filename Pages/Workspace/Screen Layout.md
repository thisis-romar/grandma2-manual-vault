# Screen Layout

This topic provides information on the screen layouts of the different screens.

## Screen 1

The screen 1 or command screen is structured in three 3 areas starting at the bottom and ending at the top.

![](../../img/img_screen1_surface_3_2.png)

_Screen Layout of Screen 1 - Command Screen_

1.  [X-Key Labels](X-Key%20Labels.md) to label X-Keys.
2.  [Command Line](Command%20Line.md) to enter keywords, open the command line feedback, or to open the message center.
3.  [User Defined Area](User-Defined%20Area.md) to create and arrange windows.

* * *

## Screen 2

The screen 2 is structured in seven areas.

![](../../img/img_screen2_surface_3_2.png)

_Screen Layout of Screen 2_

1.  [Master Section](Master%20Section.md) displays the selected executor.
2.  [User Defined Area](User-Defined%20Area.md) to create and arrange windows.
3.  [Command Line](Command%20Line.md) to enter keywords, open the command line feedback, or to open the message center.
4.  [Preset Control Bar](Preset%20Control%20Bar.md) to select a preset type.
5.  [Encoder Toolbar](Encoder%20Bar.md) to control attributes.
6.  [Time Control](Time%20Control.md) to control program time and exec time.
7.  [View Buttons](View%20Buttons.md) to recall user defined window arrangements.

* * *

## Screen 3 and Screen 4 (only grandMA2 light or grandMA2 full-size)

Screen 3 and screen 4 are structured in three areas.

![](../../img/img_screen3_surface_3_2.png)

_Screen Layout of Screen 3 and Screen 4_

1.  [Mini Executor Bar](Mini%20executor%20bar.md) to get information about the content and functions of the executor.
2.  [User defined area](User-Defined%20Area.md) to create and arrange windows.
3.  [View Buttons](View%20Buttons.md) to recall user defined window arrangements.

* * *

## Screen 5 and Screen 6 (external screens)

The external screens are structured in two areas.

![](../../img/img_screen3_surface_5_2.png)

_Screen Layout of Screen 5 and Screen 6_

1.  [User defined area](User-Defined%20Area.md) to create and arrange windows.
2.  [View Buttons](View%20Buttons.md) to recall user defined window arrangements.
