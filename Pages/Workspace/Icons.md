# Icons

The following icons are used in the grandMA2 series.

## Title Bar Icons

The title bar provides you with information icons and icons with a function.

The following table explains what the icons in title bar mean.

Icon

Meaning

![](../../img/icon_ghost-red_v3-2.png)

![](../../img/icon_home_v3-2.png)

Go home to the Setup start screen.

![](../../img/icon_change-screen_v3-2.png)

Opens the change screen pop-up.

![](../../img/icon_close_v3-2.png)

Deletes the window or closes pop-ups.

![](../../img/icon_plus_v3-2.png)

![](../../img/icon_minus_v3-2.png)

*   Show
*   Power
*   Chat
