# Icons

The following icons are used in the grandMA2 series.

## Title Bar Icons

The title bar provides you with information icons and icons with a function.

The following table explains what the icons in title bar mean.

Icon

Meaning

![](../../img/icon_world_v3-2.png)

A world is active. For more information, see [World Keyword](../../Keywords/World.md) or [Worlds, Filters, and Masks](../../Sections/Worlds%20Filters%20and%20Masks.md).

![](../../img/icon_ghost_v3-2.png)

A mask is in use. For more information, see [Use Masks in the Sheets](../Worlds%20Filters%20and%20Masks/Use%20masks%20in%20the%20sheets%20and%20windows.md).

![](../../img/icon_ghost-red_v3-2.png)

A mask is assigned. For more information, see [Use Masks in the Sheets](../Worlds%20Filters%20and%20Masks/Use%20masks%20in%20the%20sheets%20and%20windows.md).

![](../../img/icon_home_v3-2.png)

Go home to the Setup start screen.

![](../../img/icon_change-screen_v3-2.png)

Opens the change screen pop-up.

![](../../img/icon_close_v3-2.png)

Deletes the window or closes pop-ups.

![](../../img/icon_plus_v3-2.png)

Opens the virtual keyboard.

![](../../img/icon_minus_v3-2.png)

Closes the virtual keyboard.

![](../../img/icon_zoom-in_v3-2.png)

Zooms in the window.

![](../../img/icon_zoom-out_v3-2.png)

Zooms out the window.

![](../../img/icon_zoom-to-fit_v3-2.png)

Adjusts the zoom in that way that the entire recorded timecode show is visible in the window.

![](../../img/icon_follow-time_v3-2.png)

Follows the time marker and sets it to the middle of the window.

![](../../img/icon_yellow-ball_3_2.png)

Opens the window options.

## Pool Object Icons

The pool objects provides you with information about the object itself.

Icon

Meaning

![](../../img/icon_yellow-lock_v3-2.png)

Pool object is locked. To unlock a pool object, see [Unlock Keyword](../../Keywords/Unlock.md).

![](../../img/icon_red-lock_v3-2.png)

Pool object is fixed locked by default. It can not be unlocked.

![](../../img/icon_information_v3-2.png)

Pool object contains additional information.

![](../../img/icon_chain-yellow_v3-2.png)

Camera pool object is linked to a camera fixture. For more information, see [Camera Pool](../Patching/Virtual%203D%20Cameras.md).

![](../../img/icon_timecode_v3-2.png)

*   Timecode slot pool object with incoming SMPTE timecode.
*   Timecode pool object linked to a timecode slot with incoming SMPTE timecode.

For more information, see [timecode](../../Sections/Timecode.md).

![](../../img/icon_note_v3-2.png)

*   Timecode slot pool object with incoming MIDI timecode.
*   Timecode pool object linked to a timecode slot with incoming MIDI timecode.

For more information, see [timecode](../../Sections/Timecode.md).

![](../../img/icon_filter_v3-2.png)

Sequence pool object with input filter. For more information, see [cues and sequences](../../Sections/Cues%20and%20Sequences.md).

![](../../img/icon_pause_v3-2.png)

*   Paused timer pool object. For more information, see [timer](../../Sections/Timer.md).
*   Paused timecode slot pool object. For more information, see [timecode](../../Sections/Timecode.md).
*   Paused timecode pool object. For more information, see [timecode](../../Sections/Timecode.md).

![](../../img/icon_play_v3-2.png)

*   Running timer pool object. For more information, see [timer](../../Sections/Timer.md).
*   Running timecode slot pool object. For more information, see [timecode](../../Sections/Timecode.md).
*   Running timecode pool object. For more information, see [timecode](../../Sections/Timecode.md).

![](../../img/icon_off_v3-2.png)

Timer pool object is off. For more information, see [timer](../../Sections/Timer.md).

![](../../img/icon_record_v3-2.png)

Running record for macro pool object. For more information, see [record keyword](../../Keywords/Record.md) or [macros](../../Sections/Macros.md).

![](../../img/icon_green-square_v3-2.png)

*   Blinking: Refresh DMX signal for a universe pool object
*   Permanent: Change DMX values for DMX channels in the universe pool object

![](../../img/icon_rdm-green_v3-2.png)

Universe pool object with running RDM. For more information, see [RDM](../../Sections/RDM.md).

![](../../img/icon_rdm-red_v3-2.png)

Universe pool object with RDM on for this universe and global RDM off. For more information, see [RDM](../../Sections/RDM.md).

## Command Line and Message Center Icons

The [command line](Command%20Line.md) and the message center provides you with information about the:

*   Network
*   Show
*   Power
*   Chat
*   CITP

Some icons can be disabled or enabled in the message center options. For more information, see [Message Center](../Other%20System%20Tools/Message%20center.md).

The following table explains what the icons in the command line and in the message center mean.
