# What is the Replay Unit (RPU)

A **grandMA2 replay unit (RPU)** is considered as a console in the system.

It has the same software and capabilities as the consoles. It is a 19" 3 Unit rack-mount version of the console, without all the user interface and physical controls. It has almost the same connectors on the rear panel as a console (with fewer USB connectors).

It functions exactly like a console in the system.

For more information see grandMA2 replay unit (RPU) [general description](../Device%20Overview/grandMA2%20replay%20unit%20%28RPU%29.md) and [technical data](../../Sections/Technical%20Data.md).
