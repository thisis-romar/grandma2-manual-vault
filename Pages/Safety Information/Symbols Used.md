# Symbols Used

The following symbols display possible danger, useful hints and information when using the device.

**Warning:**  
The warning sign displays possible injury and hazard.  

 **Restriction:**  
The restriction sign displays known limitations of functions. 

**Important:**  
The important sign displays important information during the usage of the console.

**Hint:**  
The hint sign displays additional hints when using the console.
