# Delete Effects

Delete single, several or all effects. 

*   To delete an effect in the effect pool, press Del and tap an object in the pool. 

\-or-

*   Type into the command line:

\[Channel\]> Delete Effect "number"

*   To delete several effects in the effect pool, type into the command line:

 \[Channel\]> Delete Effect "number" Thru "number"

*   To delete all effects in the effect pool, type into the command line:

\[Channel\]> Delete Effect 1 Thru
