# Remote Controlling the System

There are several ways to remote control grandMA2.

This section describes them.

There is a [Remote Input](../Pages/Remote%20Control/Remote%20Inputs.md) part in the setup that allows simple DMX, Contact Closure and MIDI notes to trigger actions.

A more complex method is to use [MIDI Show Control - MSC](../Pages/Remote%20Control/MIDI%20Show%20Control%20%28MSC%29.md). This can be used to trigger cues, macros, control faders, and turn off executors.

There is a description of the [Web Remote](../Pages/Remote%20Control/Web%20Remote.md) - a browser interface that allows control and visualization of information - fixture and cue feedback.

The [Telnet](../Pages/Remote%20Control/Telnet%20Remote.md) connection allows the execution of any command in grandMA2. This can be very useful with 3rd party controllers - for instance a house control system.

Finally this section also has a very useful [Decimal, Hex, ASCII conversion table](../Pages/Remote%20Control/Decimal%20%E2%80%93%20Hex%20Table.md).
