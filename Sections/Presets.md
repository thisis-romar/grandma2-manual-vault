# What Are Presets?

Presets are stored presettings of parameters. They allow fast access and standardized usage. 

These presettings can be:

*   **Universal:**   
    Universal settings, abbreviated as **U**, apply to all fixtures types that contain the corresponding attributes. 

*   **Global:**  
    Global settings, abbreviated as **G**, apply to the corresponding fixture types. 

*   **Selective:**  
    Selective settings, abbreviated as **S**, apply to single fixtures. 

**Hint:**  
Make sure to use presets when programing as this facilitates your work, especially when working with a show which is used in different locations and for various customers.
