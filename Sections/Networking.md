# Networking

This section is about networking in grandMA2.

There is a short description of [what is networking](../Pages/Networking/What%20is%20networking.md).

There are some useful information about setting IP addresses in [Consoles](../Pages/Networking/Set%20the%20IP%20address%20in%20the%20console.md) and in [grandMA2 onPC](../Pages/Networking/Set%20the%20IP%20address%20in%20the%20onPC.md).

MA devices communicate and connect in a session. There is a section describing all the [relevant topics about sessions](../Pages/Networking/Session%20control.md).

The network can be used to get [DMX in and out of system](../Pages/Networking/Getting%20DMX%20in%20and%20out%20of%20the%20system.md).

There is a section about [CITP](../Pages/Networking/Using%20CITP.md).

There is a topic about [PosiStageNet or PSN](../Pages/Networking/PosiStageNet%20%28PSN%29.md). It is a protocol used to exchange real-time position values.
