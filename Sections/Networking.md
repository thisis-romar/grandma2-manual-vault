# Networking

This section is about networking in grandMA2.

There is a short description of [[Pages/Networking/What is networking|what is networking]].

There are some useful information about setting IP addresses in [[Pages/Networking/Set the IP address in the console|Consoles]] and in [[Pages/Networking/Set the IP address in the onPC|grandMA2 onPC]].

MA devices communicate and connect in a session. There is a section describing all the [[Pages/Networking/Session control|relevant topics about sessions]].

The network can be used to get [[Pages/Networking/Getting DMX in and out of the system|DMX in and out of system]].

There is a section about [[Pages/Networking/Using CITP|CITP]].

There is a topic about [[Pages/Networking/PosiStageNet (PSN)|PosiStageNet or PSN]]. It is a protocol used to exchange real-time position values.
