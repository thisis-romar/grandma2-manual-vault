# Networking

This section is about networking in grandMA2.

There is a short description of [what is networking](key_network_what_is_network.html).

There are some useful information about setting IP addresses in [Consoles](key_network_set_ip_console.html) and in [grandMA2 onPC](key_network_set_ip_onpc.html).

MA devices communicate and connect in a session. There is a section describing all the [relevant topics about sessions](key_network_session.html).

The network can be used to get [DMX in and out of system](key_network_dmx_get_in_out.html).

There is a section about [CITP](key_network_citp.html).

There is a topic about [PosiStageNet or PSN](key_network_psn.html). It is a protocol used to exchange real-time position values.
