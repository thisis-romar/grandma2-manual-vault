# Other system tools

This is a collection of system windows, not described anywhere else. This is not sheets, pools or presets, but different windows to manage some overall system settings.

The topics are:

*   [Message center](../Pages/Other%20System%20Tools/Message%20center.md)
*   [Help](../Pages/Other%20System%20Tools/Help.md)
*   [Clocks](../Pages/Other%20System%20Tools/Clock.md)
*   [Sound input window](../Pages/Other%20System%20Tools/Sound%20input%20window.md)
*   [VPU pixel mapper view](../Keywords/VPU%20pixel%20mapper%20view.md)
*   [Network dimmer](../Pages/Other%20System%20Tools/Network%20Dimmer.md)
*   [Desk status](../Pages/Other%20System%20Tools/Desk%20status.md)
*   [Performance window](../Pages/Other%20System%20Tools/Performance%20window.md)
*   [System monitor](../Pages/Other%20System%20Tools/System%20monitor.md)

To access the windows, tap on an empty part of a screen and tap Other or System
