# Operate Fixtures

This chapter deals with the various ways of controlling fixtures. 

It also comprises different sorts of sheets and their usage. 

* * *

### Requirement

Before being able to control fixtures, the following requirement has to be fulfilled:

*   Fixtures have to be patched first 

For more information on how to patch fixtures see [Adding fixtures to your show](key_patch_add_fixtures.html).
