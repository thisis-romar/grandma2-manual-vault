# HalfRate Keyword

To get the HalfRate keyword in the command line type **HalfRate** or the shortcut **HalfR** in the command line.

## Description

HalfRate is a function keyword to divide the current rate by 2.

## Syntax

HalfRate \[Object-list\]

## Example

*   To halves the rate of executor 5 type in the command line:

\[Channel\]> HalfRate Executor 5
