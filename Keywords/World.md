# World Keyword

To go to the world keyword, type **World** in the command line.

## Description

The world keyword call worlds along with its filter, and limit the access to the parameters in the word.  
World 1 is fixed by default and includes all parameters (fixtures and attributes) in the show.

World is an object keyword.

## Syntax

World \[ID\]

## Examples

Call a world.

\[Channel\]> World 4

Label a world.

\[Channel\]> Label World 4 "All Fixtures"
