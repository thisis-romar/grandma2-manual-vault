# CrashLogList keyword

**CrashLogList** is a function used to list existing crash log files in command line response window.

To execute this keyword type **CrashLogList** or the shortcut **Cr** into the command line. 

## Syntax

CrashLogList

## Example

\[Channel\]> ​CrashLogList

If there are any crash log files, these are listed in the command line response window.
