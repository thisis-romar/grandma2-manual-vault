# Interleave keyword

To execute the keyword Interleave, type **Interleave** or the shortcut **Int** into the command line. 

## Description

Interleave is used to set ranges in the MAtricks or to create a virtual array of the fixtures selected. 

The keyword Interleave is a synonym for the keyword MAtricksInterleave.

For more information see [MAtricksInterleave](MAtricksInterleave.md).

## Syntax

Interleave \[Number\]

## Example

\[Channel\]> Interleave 10

If you selected from fixture 11 to 100, pressing the key Next selects fixture 11, 21, 31 etc.
