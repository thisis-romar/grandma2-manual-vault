# Item3D keyword

To enter the keyword into the command line, type **Item3D** or the shortcut **It** into the command line. 

## Description

Item3D is an object keyword used to modify 3D objects in grandMA 3D. 

For more information see [3D objects](ma_3d/key_3dobjects.html).

## Syntax

\[Assign\] Item3D \[ID\]

\[Delete\] Item3D \[ID\]

## Example

\[Channel\]> Delete Item3D Thru

Deletes all unused 3D objects in a show.
