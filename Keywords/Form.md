# Form Keyword

To execute the keyword Form type **Form** or the shortcut **Fo** into the command line.

## Description

Form is an object type with shapes used for dynamic effects.

Forms are used as source for effects to generate dynamic values.

**Hint:**  
Forms are generated and edited in the form pool. 

## Syntax

Form \[ID\]

Form \[ID\].\[subID\]

## Example

\[Channel\]> At Form 10

Sets the first attribute of the current feature to Form 10, "Ramp Plus".
