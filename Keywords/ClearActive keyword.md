# ClearActive keyword

**ClearActive** is a function used to inactivate all values in programmer. 

To execute this keyword type **ClearActive** or the shortcut **ClearAc** into the command line. 

## Syntax

ClearActive

The ClearActive function will inactivate any active values in the programmer.

Press the button Clear twice to inactivate all values. 

For more information on further functions of the keyword [[Keywords/Clear keyword|Clear]] see [[Keywords/ClearAll keyword|ClearAll]] and [[Keywords/ClearSelection keyword|ClearSelection]].
