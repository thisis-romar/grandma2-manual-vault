# IdentifyFaderModule keyword

To execute the keyword, type **IdentifyFaderModule** or the shortcut **Id**. 

## Description

IdentifyFaderModule defines which module corresponds to which wing. That is, the wing starts to flash. The flashing automatically stops after approximately 10 seconds.

## Syntax

IdentifyFaderModule \[Number\]

## Example

\[Channel\]> IdentifyFaderModule 1

 Wing 1 starts to flash.
