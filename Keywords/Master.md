# Master Keyword

To get the Master keyword in the command line, type **Master** or the shortcut **Mas** in the command line.

## Description

The Master keyword is a function keyword to assign an executor the function master. Master is a function that will scale the output of intensity-values of an executor, according to the position of the fader.

## Syntax

**Assign** **Master** (**At**) \[Object-list\]

## Example

*   To assign executor 28 as a master, type in the command line:

\[Channel\]> Assign Master Exec 28
