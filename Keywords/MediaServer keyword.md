# MediaServer keyword

To enter the keyword, type **MediaServer** or the shortcut **Med** into the command line. 

## Description 

The keyword MediaServer is an object keyword and is used in connection with CITP. 

For more information see [Streaming CITP](../Pages/Networking/Streaming%20CITP%20video.md). 

## Syntax

MediaServer \[Id\]

## Example

\[Channel\]> List MediaServer Thru

Displays all defined CITP Mediaservers in the Command Line Feedback window
