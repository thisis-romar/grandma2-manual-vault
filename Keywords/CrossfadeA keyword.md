# CrossfadeA keyword

**CrossfadeA** is a function assigned for executors.

To execute this keyword, type **CrossfadeA** into the command line. 

## Syntax

Assign CrossfadeA (At) \[Executor-list\]

**CrossfadeA** **\[value\] \[Executor-list\] (Fade \[seconds\])**

CrossfadeA is a function that gradually fades down dimmer attributes of the current step of an executor in accordance with the position of the fader.

Depending on the executor option AB/Split this function will either act as a Crossfader for decreasing values or as a Master for the current cue (for dimmer attributes).

## Example:

\[Channel\]> ​Assign CrossfadeA At Executor 1

Assigns fader 1 the functionality of CrossfadeA.
