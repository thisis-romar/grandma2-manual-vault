# Model Keyword

To get the Model keyword in the command line, type **Model** or the shortcut **Mod** in the command line.

## Description

The Model keyword is an object keyword to access 3D models.

## Syntax

\[Function\] Model

## Examples

*   To list all 3D model of the show file in the command line feedback window, type in the command line:

\[Channel\]> List Model

*   To assign a 3D model to a fixture, type in the command line:

\[Channel\]> Assign Model 2 At Item3d 10
