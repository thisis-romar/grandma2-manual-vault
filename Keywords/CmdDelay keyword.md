# CmdDelay keyword

**CmdDelay** is a helping keyword used to indicate delay times for the cmd link.

To execute this keyword type **CmdDelay** or the shortcut **CmdD** into the command line. 

## Syntax

CmdDelay \[Value-list\]

As a helping keyword for programming functions (e.g., Store) this keyword sets the cmd delay time of the object in question.

For more information see key [Time](../Keys/Time%20Key.md). 

## Example

\[Channel\]> ​Store Cue 3 CmdDelay 4

Creates cue 3 and sets its cmd delay time to 4 seconds.
