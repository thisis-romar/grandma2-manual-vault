# Up Keyword

To go to the Up keyword type **Up** in the command line.

## Description

The Up keyword scrolls one page up in focused window. It is the same as press the Up key on the console. See [Up Key](../Keys/Up%20Key.md).

The Up keyword is a function keyword.

## Syntax

Up

## Example

Scroll one page up in the fixtures sheet.

\[Channel\]> Up
