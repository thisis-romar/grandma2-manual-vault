# Logout Keyword

To get the Logout keyword in the command line, type **Logout** or the shortcut **Logo** in the command line.

## Description

The logout keyword is a function keyword to logout the current user and change to the guest user.

## Syntax

Logout

## Example

To logout the current user and change to the guest user, type in the command line:

\[Channel\]> Logout
