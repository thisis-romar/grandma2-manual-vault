# LeaveSession Keyword

To get the LeaveSession keyword in the command line, type **LeaveSession** or one of the shortcuts **LS** or **Le** in the command line.

## Description

The LeaveSession keyword is a function keyword to leave the current session.

## Syntax

LeaveSession

## Example

*   To leave the current session, type in the command line:

\[Channel\]> LeaveSession
