# DoubleRate keyword

**DoubleRate** is a function used to double the current rate.

To execute this keyword type **DoubleRate** or the shortcut **DoubleR** into the command line. 

## Syntax

DoubleRate \[Object-list\]

## Example:

\[Channel\]> ​DoubleRate Executor 5

Doubles the rate of Executor 5.
