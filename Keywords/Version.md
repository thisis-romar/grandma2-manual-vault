# Version Keyword

To go to the version keyword, type **Version** in the command line or use the shortcut **Ve**.

## Description

The version keyword gives you access to version details of the console or software in the commandline feedback window.

It displays the following details.

*   version number
*   build date
*   build time
*   vhash
*   showfile
*   showpath
*   current user
*   parameter count

The version keyword is a function keyword.

## Syntax

Version

## Example

To see the version details of the console or the software.

*   Type Version in the command line and press Please.
