# Down keyword

To execute the keyword Down, press the key Down, type **Down** or the shortcut **Do** into the command line. 

For more information on the key Down see the [key Down](key_key_down.html). 

## Description

The keyword Down scrolls down one page in the window you are currently in. 

Down is a function keyword. 

## Syntax

Down

## Example

\[Channel\]> Down

Scrolls down one page in the fixture sheet.
