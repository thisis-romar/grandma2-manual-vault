# AllButtonExecutors keyword

To get the AllButtonExecutors keyword in the command line type **AllButtonExecutors** or the shortcut **AllB** in the command line.

## Description

AllButtonExecutors is an alias for all button executors.

AllButtonExecutors may be used as an alias to address executor 101 and upwards, on all pages.

## Syntax

\[Function\] AllButtonExecutors

## Example

*   To turn off all button executors type in the command line:

\[Channel\]> Off AllButtonExecutors
