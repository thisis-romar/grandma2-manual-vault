# Setup Keyword

To get the Setup keyword in the command line, type **Setup** or the shortcut **Set** in the command line.

## Description

The Setup keyword opens or closes the setup menu.

![](../../img/menu_setup_v3.4.png)

_Open Setup_

**Hint:**  
Another way to open or close the Setup is, press the Setup key.

The Setup keyword is a function keyword.

## Syntax

Setup

## Example

Open the setup menu.
