# MAtricksWings Keyword

To get the MAtricksWings keyword in the command line type **MAtricksWings** or one of the shortcuts **MW** or **MAtricksW** in the command line.

## Description

MAtricksWings is a function keyword to set the MAtricks Wings value.

With the MAtricksWings the fixture selection is splitted in x parts and each part is mirroring the previous part.

For more information see [[Sections/MAtricks]].

## Syntax

MAtricksWings \[parts\]

MAtricksWings + | -

MAtricksWings Off

## Example

To divide the fixture selection in two parts and the second part mirrors the adjusted values type in the command line:

\[Channel\]> MatricksWings 2
