# Login Keyword

To get the Login keyword in the command line, type **Login** or the shortcut **Log** in the command line.

## Description

The Login keyword is a function keyword to login another user.

## Syntax

**Login**

**Login** "User-name" "password"

## Example

To logout the current user and login as user Jimmy Page with the password mac, type in the command line:

\[Channel\]> Login "Jimmy Page" "mac"
