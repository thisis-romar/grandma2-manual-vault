# CrashLogDelete keyword

**CrashLogDelete** is a function used to delete crash log files on the internal drive.

To execute this keyword type **CrashLogDelete** or the shortcut **CrashLogD** into the command line. 

## Syntax

CrashLogDelete

## Example

\[Channel\]> ​CrashLogDelete

If there are any crash log  files on the internal drive, these are deleted.
