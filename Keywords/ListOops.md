# ListOops Keyword

To get the ListOops keyword in the command line, type **ListOops** or the shortcut **ListO** in the command line.

## Description

The ListOops keyword is a function keyword to get the Oops list in the command line feedback window displayed.

The Oops list displays all oopsable actions.

## Syntax

ListOops

## Example

*   To get all oopsable actions in the command line feedback window displayed, type in the command line:

\[Channel\]> ListOops
