# Pause (large) Key

The large key    Pause    is located above the large key    Go -   .

![](../../img/img_pause-key-large_v3-2.png)

_Location k__ey_ _Pause (large)_

*   Pressing    Pause    executes the [DefGoPause keyword](../Keywords/DefGoPause%20keyword.md). The executors selected will be paused.

\[time\] : Macro : DefGoPause

*   Pressing    Pause    again executes the [DefGoPause keyword](../Keywords/DefGoPause%20keyword.md) again. The pause of the selected executors ends.

\[time\] : Macro : DefGoPause
