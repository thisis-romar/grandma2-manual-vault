# Temp Key

The key Temp is located in the command area on the left of the key Top.

![](../../img/img_temp_key_3_2.png)

_Location Key Temp_ 

*   Pressing Temp enters the [Temp keyword](../Keywords/Temp.md) into the command line.

\[Channel\]> Temp

*   Pressing Temp Temp enters the [Toggle keyword](../Keywords/Toggle.md) into the command line.

\[Channel\]> Toggle
