# Copy Key

The key Copy is located in the command area below the key Del.

![](../../img/img_copy_key_3_2.png)

_Location k__ey_ _Copy_ 

*   Pressing Copy enters the [Copy keyword](../Keywords/Copy%20keyword.md) into the command line.

\[Channel\]> Copy

*   Pressing Copy Copy enters the [Clone keyword](../Keywords/Clone%20keyword.md) into the command line.

\[Channel\]> Clone
