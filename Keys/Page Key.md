# Page Key

The key Page is located in the command area on the left of the key Macro.

![](../../img/img_page-key_v3-2.png)

_Location k__ey_ _Page_ 

*   Pressing Page enters the [Page keyword](../Keywords/Page.md) into the command line.

\[Channel\]> Page

*   Pressing Page Page enters the [ChannelPage keyword](../Keywords/ChannelPage%20keyword.md) into the command line.

\[Channel\]>ChannelPage

*   Pressing Page Page Page enters the [FaderPage keyword](../Keywords/FaderPage.md) into the command line.

\[Channel\]>FaderPage

*   Pressing Page Page Page Page enters the [ButtonPage keyword](../Keywords/ButtonPage%20keyword.md) into the command line.

\[Channel\]>ButtonPage

*   Pressing and holding List + Page opens the temporary Pages Executor pool on screen 2 (except for grandMA2 onPC without MA onPC wing).
