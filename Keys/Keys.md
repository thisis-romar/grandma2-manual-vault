# \[Numeric\] Keys

The numeric keys 0 trough 9 are located in the command area of the console.

![](../../img/img_numeric_keys_3_2.png)

_Location numeric keys_

*   Pressing a numeric key enters the number in the line focused.
*   Pressing and holding ![ma](../../img/ma_1.png) + a numeric key, for example 2, selects the respective preset type for the number entered in the preset control bar.  
     
    
    ![](../../img/img_select-preset-type_v3-2.png)
    
    _Select preset type in the preset control bar_
    

\[time\] : Executing: PresetType "POSITION"

*   Pressing ![ma](../../img/ma_1.png) + a numeric key several times changes the attributes in the encoder bar to the selected preset control bar.  
     
    
    ![](../../img/img_select-attribute_v3-2.png)
    
    _Select attribute in the encoder bar_
    

\[time\] : Executing: Attribute "MP\_TR\_X"
