# Top Key

The key Top is located in the command area on the right of the key Temp.

![](../../img/img_top_key_3_2.png)

_Location Key Top_ 

*   Pressing Top enters the [Top keyword](../Keywords/Top.md) into the command line.

\[Channel\]> Top

*   Pressing Top Top enters the [Kill keyword](../Keywords/Kill.md) into the command line.

\[Channel\]> Kill
