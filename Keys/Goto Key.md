# Goto Key

The key Goto is located in the command area on the right of the key Effect.

![](../../img/img_goto_key_3_2.png)

_Location k__ey_ _Goto_ 

*   Pressing Goto enters the [Goto keyword](../Keywords/Goto.md) into the command line.

\[Channel\]> Goto

*   Pressing Goto Goto enters the [Load keyword](../Keywords/Load.md) into the command line.

\[Channel\]> Load

*   Pressing and holding ![ma](../../img/ma_1.png) + Goto enters the [LoadNext keyword](../Keywords/LoadNext.md) into the command line.

\[Channel\]> LoadNext

*   Pressing and holding ![ma](../../img/ma_1.png) + Goto Goto enters the [LoadPrev keyword](../Keywords/LoadPrev%20keyword.md) into the command line.

\[Channel\]> LoadPrev
