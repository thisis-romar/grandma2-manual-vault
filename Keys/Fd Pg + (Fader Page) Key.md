# Fd Pg + (Fader Page) Key

The key Fd Pg + is located on the left of the two 100 mm faders.

![](../../img/img_fader-page-plus-key_v3_2.png)

_Location k__ey_ _Fd Pg +_ 

Pressing Fd Pg + executes the [FaderPage keyword](../Keywords/FaderPage.md). The fader page switches to the next page.

If you are on a channel page, press Fd Pg + to open the fader pages.
