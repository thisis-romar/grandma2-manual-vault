# Up Key

The key Up is located in the command area above the key Set.

![](../../img/img_up_key_3_2.png)

_Location_ _Key_ _Up_ 

*   If you press the key Up, the scroll bar of the window or pop-up, in which the focus is set, moves up.
*   If you press and hold ![ma](../../img/ma_1.png) + Up, the monitor wing of the console moves up.
