# Store Key

The key Store is located in the command area on the left of the key

![ma](../../img/ma_1.png).

![](../../img/img_store_key_3_2.png)

_Location_ _Key_ _Store_ 

*   Pressing Store enters the [Store keyword](key_keyword_store.html) into the command line and changes the encoder bar to the [store encoder bar](key_ws_encoder_bar.html).

\[Channel\]> Store

![](../../img/img_enoder_bar_store_3_2.png)

_Store Encoder Bar_

*   Pressing and holding Store opens the pop-up Temporary Store Options.

![](../../img/popup_store_options_3_2.png)

_Pop-up Temporary Store Options_ 

  
 

*   Pressing and holding ![ma](../../img/ma_1.png) and Store enters the [Record keyword](key_keyword_record.html) into the command line.

\[Channel\]> Record
