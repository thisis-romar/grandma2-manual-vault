# Move Key

The key Move is located in the command area on the right of the key Group.

![](../../img/img_move-key_v3-2.png)

_Location k__ey_ _Move_ 

*   Pressing Move enters the [[Keywords/Move|Move keyword]] into the command line.  
    As long as the keyword Move is in the command line you can move pool objects by  
    pressing and holding a pool object and move it with your finger to a new position of the pool object   
    \- or -  
    pressing and holding the key Move and moving pool objects by tapping a pool object and then tapping the new position of the pool object.

\[Channel\]> Move

*   Pressing Move Move enters the [[Keywords/Replace|Replace keyword]] into the command line.

\[Channel\]> Replace

*   Pressing Move Move Move enters the [[Keywords/Insert|Insert keyword]] into the command line.

\[Channel\]> Insert

*   Pressing and holding ![ma](../../img/ma_1.png) + Move enters the [[Keywords/Search|Search keyword]] into the command line.

\[Channel\]> Search
