# Solo Key

The key Solo is located in the command area on the right of the key Highlt.

![](../../img/img_solo_key_3_2.png)

_Location_ _Key_ _Solo_ 

Pressing Solo executes the [Solo keyword](../Keywords/Solo.md) and turns the solo mode on or off.

The command line stays unaffected.

To disable the Solo key function, see [User Settings](../Pages/Single%20and%20Multi%20User%20Systems/User%20specific%20settings.md).
