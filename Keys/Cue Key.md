# Cue Key

The key Cue is located in the command area on the right of the key Sequ.

![](../../img/img_cue_key_3_2.png)

_Location k__ey_ _Cue_ 

*   Pressing Cue enters the [Cue keyword](../Keywords/Cue%20keyword.md) into the command line.

\[Channel\]> Cue

*   Pressing Cue Cue enters the [Part keyword](../Keywords/Part.md) into the command line.
