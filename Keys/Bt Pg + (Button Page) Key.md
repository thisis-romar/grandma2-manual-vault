# Bt Pg + (Button Page) Key

The key Bt Pg + is located on the left of the 100 mm fader.

![](../../img/img_button_page_plus_key_3_2.png)

_Location key Bt_ _Pg +_ 

*   Pressing Bt Pg + executes the [ButtonPage command](../Keywords/ButtonPage%20keyword.md). The button page switches to the next page.
