# Bt Pg - (Button Page) Key

The key Bt Pg - is located left the 100 mm fader.

![](../../img/img_button_page_minus_key_3_2.png)

_Location key Bt Pg -_

*   Pressing Bt Pg - executes the [ButtonPage command](../Keywords/ButtonPage%20keyword.md). That is: jumping to the previous ButtonPage.   
     
*   If you press and hold Bt Pg -, you will automatically jump to the first ButtonPage.
