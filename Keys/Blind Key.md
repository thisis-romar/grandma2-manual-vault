# Blind Key

The key Blind is located in the command area on the left of the key Freeze.

![](../../img/img_blind_key_3_2.png)

_Location k__ey_ _Blind_ 

Pressing Blind executes the [Blind keyword](../Keywords/Blind%20keyword.md). It toggles Blind on or off.

Pressing and holding Blind for 2 seconds executes [BlindEdit keyword](../Keywords/BlindEdit%20keyword.md). It toggles BlindEdit on or off.

To change the function of the Blind key see [User Settings](../Pages/Single%20and%20Multi%20User%20Systems/User%20specific%20settings.md).
