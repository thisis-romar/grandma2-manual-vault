# Help Key

The key Help is located on the right of the key Backup.

![](../../img/img_help_key_3_2.png)

_Location k__ey_ _Help_ 

*   Pressing Help enters the [[Keywords/Help|Help keyword]] into the command line.

\[Channel\]> Help

*   Pressing Help Help enters the [[Keywords/CmdHelp keyword]] into the command line.

\[Channel\]> CmdHelp

*   Pressing and holding Help + pressing any other key on the console, for example Fixture, opens the [[Pages/Other System Tools/Help|help window]] to the key pressed.
