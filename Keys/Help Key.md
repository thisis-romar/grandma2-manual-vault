# Help Key

The key Help is located on the right of the key Backup.

![](../../img/img_help_key_3_2.png)

_Location k__ey_ _Help_ 

*   Pressing Help enters the [Help keyword](key_keyword_help.html) into the command line.

\[Channel\]> Help

*   Pressing Help Help enters the [CmdHelp keyword](key_keyword_cmdhelp.html) into the command line.

\[Channel\]> CmdHelp

*   Pressing and holding Help + pressing any other key on the console, for example Fixture, opens the [help window](key_ost_help_viewer.html) to the key pressed.
