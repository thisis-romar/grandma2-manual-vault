# Preset Key

The key Preset is located in the command area on the right of the key Macro.

![](../../img/img_preset-key_v3-2.png)

_Location k__ey_ _Preset_

  
 

*   Pressing Preset enters the [Preset keyword](../Keywords/Preset.md) into the command line.

\[Channel\]> Preset

*   Pressing Preset Preset enters the [Attribute keyword](../Keywords/Attribute%20keyword.md) into the command line.

\[Channel\]> Attribute

*   Pressing Preset Preset Preset enters the [Gel keyword](../Keywords/Gel%20keyword.md) into the command line.

\[Channel\]> Gel

*   Pressing and holding ![ma](../../img/ma_1.png) + Preset enters the [PresetType keyword](../Keywords/PresetType.md) into the command line.

\[Channel\]> PresetType

*   Pressing and holding List + Preset opens the temporary preset pool on screen 2 for the selected preset in the preset control bar.  
     
    
    ![](../../img/window_temp-preset-pool_v3-2.png)
    
    _Temporary preset pool_
