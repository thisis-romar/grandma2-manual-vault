# \[Plus\] + Key

The key + is located next to the numeric keys in the command area.

![](../../img/img_plus_key_3_2.png)

_Location k__ey_ _\[Plus\] +_ 

*   Pressing + enters [\+ \[Plus\] keyword](key_keyword_plus.html) into the command line.

\[Channel\]> +

*   Pressing + + adds 10 % of dimmer value to the selected fixture or channel.

Macro: At +10
