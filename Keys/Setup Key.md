# Setup Key

The key Setup is located in the command area on the right of the key Tools.

![](../../img/img_setup-key_v3-2.png)

_Location_ _Key_ _Setup_ 

Pressing Setup executes the [Setup keyword](../Keywords/Setup.md) and opens the setup on the screen which was opened last.
