# Group Key

The key Group is located in the command area on the right of the key Fixture.

![](../../img/img_group_key_3_2.png)

_Location k__ey_ _Group_ 

*   Pressing Group enters the [Group keyword](../Keywords/Group.md) into the command line.

\[Channel\]> Group

*   Pressing Group for about 2 seconds opens the [master view](../Pages/Advanced%20Executor%20Functionality/Masters%20Window.md).

*   Pressing Group Group enters the [World keyword](../Keywords/World.md) into the command line.

\[Channel\]> World

*    Pressing Group Group Group enters the [Filter keyword](../Keywords/Filter.md) into the command line.

\[Channel\]> Filter

*   Pressing ![ma](../../img/ma_1.png) + Group enters the [Layout keyword](../Keywords/Layout.md) into the command line.

\[Channel\]> Layout

*   Pressing ![ma](../../img/ma_1.png) + Group Group enters the [SearchResult keyword](../Keywords/SearchResult.md) into the command line.

\[Channel\]> SearchResult

*   Pressing ![ma](../../img/ma_1.png) + Group Group Group enters the [Mask keyword](../Keywords/Mask.md) into the command line.

\[Channel\]> Mask
