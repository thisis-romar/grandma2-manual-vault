# Group Key

The key Group is located in the command area on the right of the key Fixture.

![](../../img/img_group_key_3_2.png)

_Location k__ey_ _Group_ 

*   Pressing Group enters the [Group keyword](key_keyword_group.html) into the command line.

\[Channel\]> Group

*   Pressing Group for about 2 seconds opens the [master view](key_adv_exec_master_window.html).

*   Pressing Group Group enters the [World keyword](key_keyword_world.html) into the command line.

\[Channel\]> World

*    Pressing Group Group Group enters the [Filter keyword](key_keyword_filter.html) into the command line.

\[Channel\]> Filter

*   Pressing ![ma](../../img/ma_1.png) + Group enters the [Layout keyword](key_keyword_layout.html) into the command line.

\[Channel\]> Layout

*   Pressing ![ma](../../img/ma_1.png) + Group Group enters the [SearchResult keyword](key_keyword_searchresult.html) into the command line.

\[Channel\]> SearchResult

*   Pressing ![ma](../../img/ma_1.png) + Group Group Group enters the [Mask keyword](key_keyword_mask.html) into the command line.

\[Channel\]> Mask
