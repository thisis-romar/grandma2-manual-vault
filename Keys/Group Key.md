# Group Key

The key Group is located in the command area on the right of the key Fixture.

![](../../img/img_group_key_3_2.png)

_Location k__ey_ _Group_ 

*   Pressing Group enters the [[Keywords/Group|Group keyword]] into the command line.

\[Channel\]> Group

*   Pressing Group for about 2 seconds opens the [[Pages/Advanced Executor Functionality/Masters Window|master view]].

*   Pressing Group Group enters the [[Keywords/World|World keyword]] into the command line.

\[Channel\]> World

*    Pressing Group Group Group enters the [[Keywords/Filter|Filter keyword]] into the command line.

\[Channel\]> Filter

*   Pressing ![ma](../../img/ma_1.png) + Group enters the [[Keywords/Layout|Layout keyword]] into the command line.

\[Channel\]> Layout

*   Pressing ![ma](../../img/ma_1.png) + Group Group enters the [[Keywords/SearchResult|SearchResult keyword]] into the command line.

\[Channel\]> SearchResult

*   Pressing ![ma](../../img/ma_1.png) + Group Group Group enters the [[Keywords/Mask|Mask keyword]] into the command line.

\[Channel\]> Mask
