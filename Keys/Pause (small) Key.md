# Pause (small) Key

The small key Pause is located in the command area on the right of the small key Go -.

![](../../img/img_pause-key-small_v3-2.png)

_Location k__ey_ _Pause (small)_

*   Pressing Pause enters the [Pause keyword](../Keywords/Pause.md) into the command line.

\[Channel\]> Pause

*   Pressing Pause Pause enters the [Park keyword](../Keywords/Park.md) into the command line.

\[Channel\]> Park
