# View Key

The key View is located in the command area on the left of the key Effect.

![](../../img/img_view_key_3_2.png)

_Location_ _Key_ _View_ 

*   Pressing View, enters the [View keyword](../Keywords/View.md) into the command line.

\[Channel\]> View

*   Pressing View View, enters the [ViewButton keyword](../Keywords/ViewButton.md) into the command line.

\[Channel\]>ViewButton

*   Pressing View View View, enters the [ViewPage keyword](../Keywords/ViewPage.md) into the command line.

\[Channel\]>ViewPage

*   Pressing and holding ![ma](../../img/ma_1.png) and View, enters the [Screen keyword](../Keywords/Screen.md) into the command line.

\[Channel\]>Screen
