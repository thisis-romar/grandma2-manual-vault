# If Key

The key If is located in the command area on the left of the key At.

![](../../img/img_if-key_v3-2.png)

_Location k__ey_ _If_ 

If the command line is empty,

*   pressing If enters the [IfOutput keyword](../Keywords/IfOutput.md) 

\[Channel\]> IfOutput

*   pressing If If enters the [IfActive keyword](../Keywords/IfActive.md) 

\[Channel\]> IfActive

*   pressing If If If enters the [IfProg keyword](../Keywords/IfProg.md) 

\[Channel\]> IfProg

*   pressing If If If If enters the [If keyword](../Keywords/If.md) 

\[Channel\]> If

into the command line. 

If the command line contains previous input, the If key toggles.

*   Pressing If enters the [If keyword](../Keywords/If.md) into the command line. 
*   Pressing If again enters the [EndIf keyword](../Keywords/EndIf.md) into the command line. 

\[Channel\]> Store If Group 5 EndIf
