# Time Key

The key Time is located in the command area on the left of the key Esc.

![](../../img/img_time-key_v3-2.png)

_Location_ _Key_ _Time_ 

* * *

*   Pressing Time enters the [Fade keyword](../Keywords/Fade.md) into the command line and changes the encoder bar to **Fade**.  
     
    
    ![](../../img/img_encoder-bar-fade_v3_2.png)
    
    _Encoder Bar Fade_
    
      
     

\[Channel\]> Fade 

*   If **Auto** is selected in the layer control bar of the fixture or channel sheet, pressing Time changes to the **Fade Layer**.  
     
    
    ![](../../img/img_layer-control-bar-fade-layer_v3_2.png)
    
    _Layer Control Bar - Fade Layer_
    
      
     

* * *

*   Pressing Time Time enters the [Delay keyword](../Keywords/Delay%20keyword.md) into the command line and changes the encoder bar to **Delay**.  
     
    
    ![](../../img/img_encoder-bar-delay_v3_2.png)
    
    _Encoder Bar Delay_
    
      
     

\[Channel\]>Delay

*   If **Auto** is selected in the layer control bar of the fixture or channel sheet, pressing Time Time changes to the **Delay Layer**.  
     
    
    ![](../../img/img_layer-control-bar-delay-layer_v3_2.png)
    
    _Layer Control Bar - Delay Layer_
    

* * *

*   Pressing and holding Time for about 2 seconds changes the encoder bar to **Value**.  
     
    
    ![](../../img/img_encoder-bar-value_v3_2.png)
    
    _Encoder Bar Value_
    
*   If **Auto** is selected in the layer control bar of the fixture or channel sheet, pressing and holding Time for about 2 seconds changes to the **Values Layer**.  
     
    
    ![](../../img/img_layer-control-bar-value-layer_v3_2.png)
    
    _Layer Control Bar - Values Layer_
    
      
     

* * *

If you enter [Store](../Keywords/Store.md) or [Assign](../Keywords/Assign%20keyword.md) keyword into the command line,

*   pressing then Time enters the [Fade keyword](../Keywords/Fade.md) into the command line.

\[Channel\]> Store Fade

*   pressing then Time Time enters the [OutFade keyword](../Keywords/OutFade.md) into the command line.

\[Channel\]> Store OutFade

*   pressing then Time Time Time enters the [Delay keyword](../Keywords/Delay%20keyword.md) into the command line.

\[Channel\]> Store Delay

*   pressing then Time Time Time Time enters the [OutDelay keyword](../Keywords/OutDelay.md) into the command line.

\[Channel\]> Store OutDelay

*    pressing Time Time Time Time Time enters the [SnapPercent keyword](../Keywords/SnapPercent.md) into the command line.

\[Channel\]> Store SnapPercent

*   pressing then Time Time Time Time Time Time enters the [CmdDelay keyword](../Keywords/CmdDelay%20keyword.md) into the command line.

\[Channel\]> Store CmdDelay

* * *

*    Pressing and holding ![ma](../../img/ma_1.png) + Time opens the **Effect Layer** in the encoder bar.  
     
    
    ![](../../img/img_encoder-bar-effect_v3_2.png)
    
    _Encoder Bar - Effect Layer_
    
      
      
     
*   Pressing and holding ![ma](../../img/ma_1.png) + Time again, changes to the next **Effect Layer**.  
     
*   If **Auto** is selected in the layer control bar of the fixture or channel sheet, pressing and holding ![ma](../../img/ma_1.png) + Time changes to the next **Effect Layer**.  
     
    
    ![](../../img/img_layer-control-bar-effect-layer_v3_2.png)
    
    _Layer Control Bar - Effect Layer_
