# MA Key

The ![ma](../../img/ma_1.png) key is located in the command area below the 0.

![](../../img/img_ma-key_v3-2.png)

_Location_ _key_ _MA_ 

Press ![ma](../../img/ma_1.png) to see all active icons of the message center in the command line. This command shows the icons even if the status is set to No (= empty) in the message center options. For more information on the icons of the message center see [icons](../Pages/Workspace/Icons.md). For more information on the message center see [message center](../Pages/Other%20System%20Tools/Message%20center.md).
