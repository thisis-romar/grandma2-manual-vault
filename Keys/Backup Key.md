# Backup Key

The key Backup is located on the right of the key Setup.

![](../../img/img_backup_key_3_2.png)

_Location k__ey_ _Backup_ 

Pressing Backup executes [Backup keyword](key_keyword_backup.html) and opens the [Backup Menu](key_backup_menu.html).

Pressing Backup Backup executes [SaveShow keyword](key_keyword_saveshow.html) and saves the show data on the currently selected drive.
