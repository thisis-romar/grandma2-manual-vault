# Backup Key

The key Backup is located on the right of the key Setup.

![](../../img/img_backup_key_3_2.png)

_Location k__ey_ _Backup_ 

Pressing Backup executes [Backup keyword](../Keywords/Backup.md) and opens the [Backup Menu](../Sections/Backup%20Menu.md).

Pressing Backup Backup executes [SaveShow keyword](../Keywords/SaveShow.md) and saves the show data on the currently selected drive.
