# B.O. (Blackout) Key

The key B.O. is located in the command area on the right of the key Align.

![](../../img/img_bo-key_v3-2.png)

_Location k__ey_ _B.O._ 

As long as you press and hold B.O., the blackout mode is on. For more information see [Blackout keyword](key_keyword_blackout.html).

The command line stays unaffected.

To disable the B.O. key see [User Settings](key_single_multi_systems_user_settings.html).
