# Next Key

The key Next is located in the command area on the right of the key Set.

![](../../img/img_next-key_v3-2.png)

_Location k__ey_ _Next_ 

*   If a pop-up dialog is open, for example the pop-up Choose store method, you can set the focus on the following options by pressing the key Next.  
     
*   Pressing Next executes the [Next keyword](../Keywords/Next.md). The fixture with the lowest fixture ID is selected.

\[time\] : Executing : Next

*   Pressing and holding ![ma](../../img/ma_1.png) + Next executes the [NextRow keyword](../Keywords/NextRow.md). For more information see [MAtricks](../Sections/MAtricks.md).

\[time\] : Executing : NextRow

*   Pressing Next + Prev executes the [All keyword](../Keywords/All.md).

\[time\] : Executing : All
