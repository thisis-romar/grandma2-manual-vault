# Macro Key

The key Macro is located in the command area on the right of the key Page.

![](../../img/img_macro-key_v3-2.png)

_Location k__ey_ _Macro_ 

*   Pressing Macro enters the [Macro keyword](../Keywords/Macro.md) into the command line.

\[Channel\]> Macro

*   Pressing Macro Macro enters the [Timecode keyword](../Keywords/Timecode.md) into the command line.

\[Channel\]> Timecode

*   Pressing Macro Macro Macro enters the [Agenda keyword](../Keywords/Agenda.md) into the command line.

\[Channel\]> Agenda

*   Pressing and holding ![ma](../../img/ma_1.png) + Macro enters the [Timer keyword](../Keywords/Timer.md) into the command line.

\[Channel\]> Timer

*   Pressing and holding ![ma](../../img/ma_1.png) + Macro Macro enters the [Plugin keyword](../Keywords/Plugin.md) into the command line.

\[Channel\]> Plugin

*   Pressing and holding List + Macro temporarily opens the Macro Pool on screen 2 (except for grandMA2 onPC without MA onPC wing).

![](../../img/window_temp-macro-pool_v3-2.png)

_Temporary Macro Pool_
